# 🐉 Dragon Hunter — Ancient Legends

A fast-paced 2D side-scrolling action game built in **C** using **SDL2**.
Fight your way through 10 brutal levels of flying and ground dragons,
collecting swords, powerballs, and coins to survive.

## 🎮 Gameplay
- Battle through **10 unique levels** — from *The Awakening* to *The Final Hunt*
- Face **flying and ground dragons** that grow stronger each level
- Pick up **swords** (5 hits each) and **powerballs** to defeat enemies
- **Double jump** across platforms to dodge attacks
- Earn **combo kills** and collect **coins** to boost your score

## ⌨️ Controls
| Key | Action |
|---|---|
| `Arrow Left / Right` | Move |
| `Space` or `Up Arrow` | Jump (double jump supported) |
| `Z` | Sword attack |
| `Q` | Fire powerball |
| `Escape` | Pause |
| `Enter` | Confirm / Next level |

## 🛠️ Built With
- **Language:** C
- **Graphics & Input:** SDL2
- **Font Rendering:** SDL2_ttf
- **Audio:** SDL2_mixer
- **Build System:** Makefile

## 🚀 How to Build & Run
### Requirements
- GCC compiler
- SDL2, SDL2_ttf, SDL2_mixer

### Build & Run
make
./DragonHunter
